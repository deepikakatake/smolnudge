# SmolNudge

SmolNudge is a fun, emoji-filled wellness app that helps users track their mood, stay on top of medication, and get a daily laugh.

## ✨ Features

- 🌤️ Daily mood check-ins
- 💊 Medication reminders
- 😂 Joke of the day
- 📈 Wellness tracking dashboard

## 🚀 Getting Started

To run locally:

```bash
npm install
npx expo start
